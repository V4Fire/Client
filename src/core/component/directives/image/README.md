# core/component/directives/image

This module provides a directive to load images by using `background-image` or `src`.

## Usage

```
< .&__not-img v-image = {
	src: 'https://fakeimg.com'
}

< img.&__img v-image = {
	src: 'https://fakeimg.com'
}
```

For more details go to [`core/dom/image`](core/dom/image/index.ts)
